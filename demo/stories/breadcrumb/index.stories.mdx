# OSUI Breadcrumb

支持所有antd属性：见[antd文档](https://ant.design/components/breadcrumb-cn/)

面包屑能够作为用户路径的记录，可点击切换至对应页面，至多展示5级面包屑内容，超过部分展示一级与末尾2级，中间省略号显示，可从后之前逐级展开内容
